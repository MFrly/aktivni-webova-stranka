    • 2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

    Napiš aspoň pět zásad ve podobě číslovaného seznamu.

    Použij angličtinu!

    1. Správné formátování: Ujistěte se, že váš kód je správně odsazený a formátovaný. To zahrnuje používání konzistentního odsazení (obvykle 2 nebo 4 mezery) a řádné zarovnání kódu.

    2. Zpracování chyb: Používejte try-catch bloky pro zachycení a zpracování výjimek. To pomůže zabránit pádu vaší aplikace a umožní lépe diagnostikovat chyby.

    3. Jednoduchost a čitelnost: Váš kód by měl být co nejjednodušší a nejčitelnější. Používejte výstižné názvy proměnných a funkcí a komentujte složité části kódu, aby bylo jasné, co dělají.

    4. Camel case je konvence pro pojmenovávání proměnných, funkcí a identifikátorů v JavaScriptu. V camel case se víceslovné názvy tvoří spojením slov bez mezer, přičemž každé slovo po prvním začíná velkým písmenem.

    5. Testování: Pravidelně testujte svůj kód, abyste odhalili a odstranili chyby. Automatizované testy jsou výborným nástrojem pro ujištění se, že váš kód funguje správně.

    4: Jak je možné v JavaScriptu deklarovat proměnné?

    Uveďte na příkladech různé datové typy, které je možné využívat v JS.

    Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

    let x = 0; // s přiřazením hodnoty
    let x; // bez přiřazení hodnoty
    let x = null; // proměnná momentálně nemá žádnou platnou hodnotu
    let x = 69; // celé číslo (typ number)
    let x = 4.20 // reálné číslo (typ number)
    let x = 2.5e3; // zápis desetinného čísla pomocí exponentu
    let x = "Hello, World!"; // typ string
    let prestupny = false // Boolean (logická hodnota)

    5: Jaké operátory můžeme používat v JavaScriptu?

    Vypište operátory, které patří do uvedených skupin.

    U méně obvyklých operátorů dopište do závorky jejich význam!

    Matematické

    +, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

    Přiřazovací

    = (přiřazení),+= (přiřazení sčítání),-= (přiřazení odečítání),*= (přiřazení násobení),/= (přiřazení dělení),%= (přiřazení modulo)

    Logické

    && (logický AND),|| (logický OR),! (logický NOT)

    Porovnávací

    == (rovno),!= (nerovno),=== (přísně rovno),!== (přísně nerovno),> (větší než),< (menší než),>= (větší nebo rovno),<= (menší nebo rovno)

    Bitové

    & (bitový AND),| (bitový OR),^ (bitový XOR),~ (bitový NOT),<< (bitový posun doleva),>> (bitový posun doprava),>>> (bitový posun doprava s nulovým rozšířením)

    Typové

    typeof (typ proměnné), instanceof (testuje, zda objekt je instancí určité třídy nebo konstruktoru)

    6: Která popup okna lze využít v JavaScriptu?

    K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

    Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

    alert()

    /* Zobrazí popup okno s upozorněním */
    alert('Nezadal jsi správné heslo!');

    prompt()

    /* Do proměnné password vloží heslo zadané uživatelem */
    let password = prompt('Zadejte své heslo:');

    confirm()

    /* Toto okno zobrazuje otázku, na kterou uživatel může odpovědět "OK" (potvrď) nebo "Cancel" (zrušit). */
    if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

    Varování: Používání nadměrných popup oken může uživatele rušit a zhoršit jejich uživatelskou zkušenost. Je důležité používat popup okna s rozvahou a pouze tam, kde jsou opravdu potřeba.
